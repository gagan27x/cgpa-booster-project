# CGPA Booster

CGPA Booster is a static multi-page student portal built with HTML, CSS, and JavaScript.

## Pages

- `index.html` - home dashboard
- `notes.html` - notes and PYQ overview
- `calculator.html` - CGPA, GPA, and percentage calculator
- `attendance.html` - subject-wise attendance tracker
- `resources.html` - searchable resource library
- `about.html` - project overview

## Folder Structure

- `css/` - shared styles and responsive rules
- `js/` - shared navigation, calculator, attendance, and search logic
- `assets/` - image and icon placeholders
- `pdf/Notes` - notes uploads
- `pdf/PYQs` - previous year question papers

## Usage

Open `index.html` in a browser and navigate through the site using the top menu.

The calculator and attendance pages are interactive and update in the browser without any build step.
